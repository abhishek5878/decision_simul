# 20 High-Quality Onboarding Flows for Benchmarking

## Selection Criteria
- Clear "first value" moment
- Known for good UX or high adoption
- Publicly accessible onboarding
- India-relevant where possible

---

## FINANCIAL / PAYMENTS / NEO-BANKING (6 products)

### 1. Razorpay
- **Category:** Fintech / Payments
- **India Relevance:** High (India-first payment gateway)
- **Why Selected:** Clear value prop, minimal friction, strong trust signals
- **Onboarding Pattern:** Quick signup → Dashboard access → First transaction setup

### 2. PhonePe
- **Category:** Fintech / Payments
- **India Relevance:** High (India's largest UPI app)
- **Why Selected:** Extremely low-friction onboarding, instant value (scan QR), minimal data collection
- **Onboarding Pattern:** Phone number → OTP → Ready to pay

### 3. CRED
- **Category:** Fintech / Credit Card Rewards
- **India Relevance:** High (India-first credit card management)
- **Why Selected:** Strong value proposition upfront, gamification, clear rewards
- **Onboarding Pattern:** Value prop → Credit card linking → Instant rewards

### 4. Groww
- **Category:** Fintech / Investment
- **India Relevance:** High (India-first investment platform)
- **Why Selected:** Educational onboarding, progressive disclosure, low commitment
- **Onboarding Pattern:** Learn → Sign up → Start investing (minimal)

### 5. Paytm
- **Category:** Fintech / Payments
- **India Relevance:** High (India's digital wallet pioneer)
- **Why Selected:** Ubiquitous in India, simple onboarding, multiple use cases
- **Onboarding Pattern:** Phone → OTP → Wallet ready

### 6. Zerodha
- **Category:** Fintech / Trading
- **India Relevance:** High (India's largest discount broker)
- **Why Selected:** Minimal onboarding, instant account creation, clear value
- **Onboarding Pattern:** Basic info → KYC → Trading ready

## CONSUMER APPS (5 products)

### 7. Swiggy
- **Category:** Consumer / Food Delivery
- **India Relevance:** High (India's largest food delivery)
- **Why Selected:** Location-first onboarding, instant value (see restaurants), minimal friction
- **Onboarding Pattern:** Location → Browse → Order (no signup required initially)

### 8. Zomato
- **Category:** Consumer / Food Delivery
- **India Relevance:** High (India's food discovery platform)
- **Why Selected:** Browse-first, signup optional, strong value before commitment
- **Onboarding Pattern:** Browse → Value seen → Sign up when ordering

### 9. Notion
- **Category:** Consumer / Productivity
- **India Relevance:** Medium (Global, but popular in India)
- **Why Selected:** Excellent onboarding with templates, progressive value, no commitment
- **Onboarding Pattern:** Templates → Try → Sign up when ready

### 10. Canva
- **Category:** Consumer / Creator Tools
- **India Relevance:** Medium (Global, growing in India)
- **Why Selected:** Instant value (design tool), free tier, no commitment
- **Onboarding Pattern:** Start designing → Value delivered → Optional signup

### 11. Spotify
- **Category:** Consumer / Entertainment
- **India Relevance:** Medium (Global, strong India presence)
- **Why Selected:** Free tier, instant value (music), signup optional
- **Onboarding Pattern:** Browse → Play → Sign up for features
